# 图标资源说明

小程序需要以下图标文件（建议尺寸 81x81 像素）：

## 标签栏图标：
- home.png - 首页图标（未选中）
- home-active.png - 首页图标（选中）
- room.png - 房间图标（未选中）
- room-active.png - 房间图标（选中）
- booking.png - 预订图标（未选中）
- booking-active.png - 预订图标（选中）

## 其他图片：
- default-room.jpg - 默认房间图片（建议尺寸 400x300 像素）

这些图标可以通过以下方式获取：
1. 使用 iconfont 网站下载
2. 使用设计工具制作
3. 从免费图标库获取

注意：实际使用时请替换为合适的图标文件。
