# 酒店预订小程序

这是一个基于微信小程序开发的酒店预订系统客户端，与后端房间管理系统集成。

## 功能特性

### 🏨 房间预订
- 查看可用房间列表
- 按房型、价格筛选房间
- 实时房间状态显示
- 房间图片和详细信息展示

### 📅 预订管理
- 选择入住和离店日期
- 填写客户信息（姓名、手机号、身份证）
- 自动计算预订金额
- 实时提交预订申请

### 📋 我的预订
- 根据手机号查询预订记录
- 查看预订状态和详情
- 支持取消预订操作
- 下拉刷新最新状态

## 页面结构

```
pages/
├── index/           # 首页 - 酒店介绍和快捷入口
├── rooms/           # 房间列表 - 浏览和选择房间
├── booking/         # 预订页面 - 填写预订信息
├── my-bookings/     # 我的预订 - 查看预订记录
└── logs/           # 日志页面 - 调试信息
```

## 工作流程

1. **客户选择房间**
   - 在房间列表页面浏览可用房间
   - 查看房间详情、价格和图片
   - 点击"立即预订"进入预订页面

2. **填写预订信息**
   - 输入客户姓名、手机号、身份证
   - 选择入住和离店日期
   - 系统自动计算总金额
   - 提交预订申请

3. **后台处理预订**
   - 预订提交后状态为"待确认"
   - 前台人员在管理系统中查看预订
   - 确认预订后状态变更为"已确认"

4. **客户查看预订**
   - 在"我的预订"页面输入手机号查询
   - 查看预订状态和详细信息
   - 可以取消未确认的预订

## 与后端集成

小程序通过以下API与房间管理后端系统集成：

- `GET /api/rooms` - 获取房间列表
- `POST /api/bookings` - 创建预订
- `GET /api/bookings?phone=xxx` - 查询预订
- `PATCH /api/bookings/{id}/status` - 更新预订状态

## 使用说明

### 开发环境
1. 确保后端服务运行在 `http://localhost:3000`
2. 使用微信开发者工具打开项目
3. 在项目设置中关闭域名校验（开发阶段）
4. 编译运行小程序

### 生产环境
1. 配置合法的服务器域名
2. 在 `utils/api.js` 中修改 BASE_URL
3. 上传代码并提交审核

## 注意事项

- 图标资源需要单独准备，参考 `/images/README.md`
- 默认房间图片路径为 `/images/default-room.jpg`
- API 请求超时时间为 60 秒
- 本地存储用于缓存客户手机号

## 自定义配置

### 修改酒店信息
在 `pages/index/index.ts` 中修改 `hotelInfo` 对象：

```typescript
hotelInfo: {
  name: '您的酒店名称',
  description: '酒店描述',
  features: ['特色1', '特色2', '特色3']
}
```

### 修改主题色彩
在各页面的 CSS 文件中修改渐变色彩：

```css
background: linear-gradient(135deg, #您的主色 0%, #您的副色 100%);
```
